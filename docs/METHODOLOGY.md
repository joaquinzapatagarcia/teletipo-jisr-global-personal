# Metodología

Cada índice parte de una base explícita. Los titulares deduplicados aportan una señal firmada según palabras de presión, contexto o alivio. El mero número de artículos neutrales no aumenta el índice.

La lectura nueva se amortigua frente a la anterior. Con menos de cuatro evidencias, cualquier movimiento queda limitado a un punto. La confianza combina cantidad de evidencias y diversidad de dominios.

La salida distingue `ultima_ejecucion`, `ultima_actualizacion_con_datos`, `edicion`, `estado_publicacion` y `estado_fuentes`. Si todas las fuentes fallan o no aportan señales, no se recalculan los índices ni se presenta una frase nueva como medición nueva.

GDELT es una fuente abierta de titulares, no la realidad completa. Estos índices son instrumentos editoriales para ordenar criterio; no son indicadores oficiales, predicciones ni recomendaciones de inversión.
